SOBRE O SISTEMA

CRUD - Jogos Eletrônicos é um sistema web moderno desenvolvido para gerenciar um catálogo de jogos. A aplicação evoluiu ao longo do tempo, recebendo melhorias em estrutura, tecnologia e experiência do usuário.

Funcionalidades

  • Cadastro completo de jogos com título, sinopse, gênero, imagem, preço e mais.
  • Visualização aberta para todos os usuários.
  • Edição e remoção disponíveis apenas para usuários logados.
  • Busca por título e filtro por categoria com resposta dinâmica.
  • Destaques e lançamentos na página inicial.
  • Redirecionamento para página de erro em acessos não autorizados.

Evoluções do Projeto

  • Conversão de páginas JSP para HTML estático com carregamento dinâmico.
  • Criação de API REST utilizando Servlets que retornam JSON.
  • Uso de fetch API para integração entre front-end e back-end.
  • Simulação de banco de dados com arquivos .json para jogos e usuários.
  • Separação entre front-end e back-end para facilitar manutenção e expansão.

Menu de Navegação

  • Home: Exibe jogos em destaque e lançamentos.
  • Sobre o Sistema: Página com informações do projeto.
  • Login: Autenticação de usuários.
  • Após login: Permite adicionar, editar e excluir jogos (conforme permissões).

Tecnologias Utilizadas

  • Back-end: Java (Servlets), GSON
  • Front-end: HTML5, CSS3, Bootstrap 5, JavaScript moderno (ES6+)
  • Dados: JSON (usuarios.json e jogos.json)
